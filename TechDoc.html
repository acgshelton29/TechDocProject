Introduction

JavaScript is an object-oriented scripting language often used to perform functionality to websites. It can also be connected to elements of a website in order to manipulate and assert control over them.

You can find standard objects included in JavaScript's basic library such as array, math, and date. It also has a basic elements like operators, statements, and control structures.

  • Client-Side Javascript allows objects control over the browser and the Document Object Model (DOM). As an example, client-side extensions allow an application to place elements on an HTML document and respond to users' interaction.
  • Server-side JavaScript creates objects that assists with Running JavaScript on a server.

Java and JavasScript

Java and JavaScript are similiar in a few ways, but are ultimately fundamentally different. For example, JavaScript does not have Java's static typing or strong type checking. JavaScript does however follow Java's basic control flow, expression syntax, and naming conventions.

Variables

A varibale acts as a container for data and the name of the variable must adhere to certain rules.

A variable in JavaScript must either start with a letter, underscore, or a dollar sign. Javascript is also case sensitive, so the letter "A" is not considered the same as the letter "a".

JavaScript will also accept ISO 8859-1 or unicode letters.

Declaring Variables

There are three ways that you can declare a variable in JavaScript:

By using the keyword, var.

var x = 7;

This way can be used on both global and local variables.

By assigning it a value.

x = 7;

This way only declares global variables, and should never be used.

By using the keyword, let.

let x = 7;

This can create a block scope local variable.

If. . . Else Statements

An If statement will execute a block of code when a certain condition is true. If you want a block of code to run when that condition is false, you add an else statement.

if (condition) {statement1;} else {statement2;}

The condition can be anything as long as it is an expression that results in a boolean value. the statements can be any type of statement as long as it is symantically and logically correct.

You can also compound statements using elseif.

if (condition1) {statement1;} else if (condition2) {statement2;}
else if (condition3) {statement3;} else {statement4;}

In the case of several conditions, only the first true one will be executed. If it becomes necessary to run multiple statements, group them together in a single block.

if (condition) { statement_1_runs_if_condition_is_true;
statement_2_runs_if_condition_is_true; }
else {statement_3_runs_if_condition_is_false;
statement_4_runs_if_condition_is_false; }

It is not advised to use assignment expressions in conditions because they can be confused with equality assignments. For example:

if (x = y) {statement;}

If you need to use an assignment expression, it is courteous to use an extra set of parantheses:

if ((x = y)) {statement;}
